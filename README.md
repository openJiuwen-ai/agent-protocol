# Agent Protocol

English | [简体中文](./README_zh.md)

## Introduction

**Agent Protocol** is a collection of software development kits (SDKs) for agent interoperability protocols. It helps developers quickly integrate agents and enable agent-to-tool / agent-to-agent interoperability across different runtime environments.

This repository currently includes three SDKs:
- Model Context Protocol (MCP) CPP SDK
- Agent2Agent Protocol (A2A) CPP SDK
- A2X agent registry
